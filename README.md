# Voice-Sample

A long voice sample for evaluating TTS voice cloning quality.

## Contents

- `26_03_2026_16_08.flac` — Voice recording (~30MB, mono 24kHz FLAC, 20m 54s)
- `transcript.md` — Timestamped transcript (AssemblyAI, 97.4% confidence)

## Purpose

Used to check how well voice cloning utilities reproduce the source voice. The FLAC is mono 24kHz — the standard input format for most TTS/voice cloning tools (Coqui, RVC, ElevenLabs, etc.).
